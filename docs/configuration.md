# Configuration

the configuration is looked for in these locations:

- `./topdata-package-util.yaml`
- `/etc/topdata-package-util.yaml`
- `$HOME/.config/topdata-package-util.yaml`


## example config file

```yaml
webserver:
  port: "8080"
  username: "test"
  password: "test"

portfolio: /topdata/topdata-package-portfolio/portfolio.yaml

```
